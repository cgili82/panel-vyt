📥 Cargar nueva base de Time Billing

Esta acción actualiza la base de todo el equipo, no solo tu vista local.
¿Quién debe hacer esta acción?
El responsable del mes (quien descargó el WIP de Time Billing). Al cargar el archivo aquí, la base se actualizará en tu panel. Luego debes subirla a OneDrive para que Fabrissio, Cristina, Katherine, Mayte y Andrea también la vean.
①
Selecciona el export de TB que descargaste de Time Billing este mes. El panel detecta las columnas automáticamente y normaliza áreas, formas y socios.
②
Los estados, comentarios y montos ya trabajados se preservan para asuntos que coincidan (mismo cliente + asunto).
③
Al terminar, el panel generará el Excel actualizado y te guiará para subirlo a OneDrive y compartirlo con todo el equipo.
⚠ Atención: los asuntos que no coincidan con la base anterior se agregarán como nuevos. Los que ya no estén en el export TB se marcarán pero no se eliminarán.
